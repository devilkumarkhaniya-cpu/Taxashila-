# **App Name**: Takshashila

## Core Features:

- Intelligent Performance Analyst: A tool that uses reasoning to analyze student test performance across subjects like Maths and Reasoning to provide personalized study recommendations.
- Advanced Mock Test Engine: Comprehensive testing environment featuring topic-wise and full-length tests with integrated timers, negative marking, and auto-submit functionality.
- Integrated PDF Library: Secure document viewer for free and premium educational resources with advanced search and category filtering capabilities.
- Dynamic Student Dashboard: Personalized user hub featuring performance charts, test history, score analytics, and leaderboard ranking.
- Current Affairs Portal: Daily, weekly, and monthly content updates with interactive quizzes to keep students informed.
- Secure Auth & Profiles: Unified authentication system supporting Email and Google login with comprehensive user profile management.
- Premium Subscription System: Razorpay-powered payment integration for managing monthly plans and granting automatic access to premium mock tests and PDFs.
- Admin Control Center: Backend management suite for uploading resources, creating MCQs, managing users, and viewing platform-wide analytics.

## Style Guidelines:

- Professional foundation using Clean White (#FFFFFF) and Deep Raven Black (#1A1A1A).
- Solar Yellow (#FFD700) accent color for primary call-to-actions, status highlights, and achievement badges.
- Modern pairing of 'Montserrat' for bold, authoritative headlines and 'Inter' for high-readability body text.
- Minimalist, sleek outline icons representing different subjects and exam metrics to maintain a clean aesthetic.
- Mobile-first responsive design utilizing an editorial grid with bento-style cards for performance data.
- Subtle micro-interactions, such as graceful fade-ins and smooth progress bar transitions during test submissions.
