# Takshashila - Competitive Exam Preparation Platform

Gujarat's leading platform for GPSC, GSSSB, SSC, and Police Constable preparation.

## Current Features

- **Secure Admin Panel**: Accessible via password `Devil@01754`. Manage study materials, exam news, and subscribers.
- **Exam News (Job Alerts)**: Real-time government job updates with "Apply Now" links and category filters.
- **AI-Powered Mock Tests**: Generate practice questions dynamically using Gemini AI.
- **Specialized Diagnostic Tests**: Includes a Blood Relation module with a 5-generation family tree visualization and 13 targeted questions.
- **Personalized Analysis**: AI analysis of student performance with study recommendations.
- **Visitor Engagement**: Floating notification bell and newsletter subscription system to collect student emails.
- **Live Analytics**: Basic visitor tracking (device type, path, timestamp) visible in the Admin Panel.
